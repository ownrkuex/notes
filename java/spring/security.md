## maven

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
</dependency>
```

  * com.springframework.security.core
  * ~.access
  * ~.authentication
* spring-security-web.jar
  * ~.web
* spring-security-config.jar
  * ~.config
